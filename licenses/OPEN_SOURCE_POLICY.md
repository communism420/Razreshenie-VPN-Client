# Open-Source политика Razreshenie VPN Client

Razreshenie VPN Client позиционируется как полностью открытое и свободное ПО.

Текущая версия приложения: 1.1.3.

- Лицензия проекта: GPLv3.
- Исходный код должен быть доступен для проверки, модификации и самостоятельной сборки.
- В приложении не должно быть телеметрии, аналитики, скрытого сбора данных или закрытых модулей.
- Профили, подписки, правила split tunneling, настройки, логи и sing-box core хранятся локально.
- Сетевые запросы должны быть понятны пользователю: загрузка sing-box, подписок, raw JSON/TXT-правил, проверка IP/DNS.
- Раздел «О проекте» должен явно приглашать сообщество к аудиту и участию.
- В документации и разделе «О проекте» должна быть сохранена атрибуция zapret-kvn: часть кода, графической архитектуры и дизайн-подходов адаптированы из https://github.com/youtubediscord/zapret-kvn.
- В документации и разделе «О проекте» должна быть сохранена благодарность russia-mobile-internet-whitelist за домены из российского «белого списка» для встроенного bypass: https://github.com/hxehex/russia-mobile-internet-whitelist.
- SVG-флаги стран для интерфейса должны сохранять атрибуцию flag-icons: https://github.com/lipis/flag-icons, лицензия MIT.
