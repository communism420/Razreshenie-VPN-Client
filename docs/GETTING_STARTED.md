# Первый запуск

Razreshenie VPN Client - Windows VPN-клиент на PyQt6 и sing-box. Все пользовательские данные хранятся локально в `%USERPROFILE%\Razreshenie VPN`.

## Установка из исходников

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt
python main.py
```

Proxy-режим работает без прав администратора. TUN-режим создаёт виртуальный сетевой интерфейс и может запросить UAC при подключении.

## Мастер первого запуска

При первом запуске приложение показывает мастер настройки. Он не подключает VPN автоматически, а только сохраняет базовые параметры.

Рекомендуемый порядок:

1. Выберите режим подключения.
2. Оставьте фоновую проверку соединения включенной, если хотите автоматическое восстановление.
3. Включайте автозапуск Windows только если клиент должен стартовать вместе с системой.
4. В поле "После мастера" выберите первое действие: импортировать сервер, добавить подписку или скачать sing-box core.

## Proxy или TUN

`Proxy` создаёт локальный mixed inbound SOCKS5/HTTP. Этот режим проще, не требует администратора и подходит для приложений, которые умеют использовать системный proxy или proxy guard.

`TUN` создаёт системный туннель. Он подходит, когда нужно направлять трафик большинства приложений через sing-box, использовать split tunneling, process-правила и Kill Switch.

## Первый сервер

Во вкладке `Серверы` можно импортировать одиночную ссылку или несколько ссылок/файлов сразу. Поддерживаются VLESS, Trojan, VMess, Hysteria2, TUIC v5, Shadowsocks и WireGuard.

После импорта:

1. Выберите сервер в таблице.
2. Нажмите подключение на главной панели.
3. При необходимости проверьте отклик выбранного сервера или всех серверов.

## Подписка

Во вкладке `Подписки` добавьте URL провайдера. При обновлении клиент делает основной запрос, при подозрении на неполный ответ пробует дополнительные запросы и использует лучший снимок. Последний успешный payload сохраняется в локальный кэш.

## Базовая проверка

Для проверки исходников без запуска GUI:

```powershell
python -m compileall -q gui core models utils main.py tests
python -m unittest discover -s tests -v
python main.py --self-check
```

## Если подключение не работает

1. Откройте вкладку `Логи`.
2. Проверьте, скачан ли sing-box core во вкладке `О проекте`.
3. В Proxy-режиме проверьте, не занят ли mixed port.
4. В TUN-режиме перезапустите приложение от имени администратора.
5. Смотрите [Troubleshooting](TROUBLESHOOTING.md).

## Обновление приложения

Во вкладке `Настройки` можно выбрать способ обновления приложения:

- `Скачать отдельно` - текущая безопасная схема: новый EXE сохраняется в папку `downloads/app-updates`.
- `Заменить текущий EXE` - для собранной Windows-версии клиент скачает новый EXE, закроется и заменит файл в папке запуска.
